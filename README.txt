fuzzybolge is a collection of tools to help with making malbolge programs.

Included files:
- chars.txt - All accepted characters and their ascii values. Important for interacting with tools.
- encoder.bat - Simple batch program, takes two variables: o (operator) and c (operator number). Outputs ascii number of character for .mal file.
- decoder.bat - Reverse of encode.bat. Takes two variables: [c] (acii value of the character) and c (operator number). Outputs the operator number.
